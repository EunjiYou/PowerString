# PowerString 프로젝트
개발 초보자의 코딩 입문을 돕기 위한 코드 프로그램
<br/><br/><br/> 

# 프로젝트 개요
- 개발인원 : 5명
- 개발 기간 : 2018.10.08 - 2018.10.15
- 개발 환경<br/>
  - 사용 언어 : C# 
  - 사용 툴 : Visual Studio, MSSQL
<br/><br/><br/>

# 주요 구현 기능
- 회원 가입
- 로그인, 로그아웃
- 모든 유저 정보 확인
- 타이핑 테스트
- 계정 탈퇴
<br/><br/><br/>

# 핵심 기술
- Database 설계
- Entity Framework를 이용한 Database 연동
- LINQ문을 이용한 GridView Data Binding
- UnitTest를 이용한 단위 테스트
<br/><br/><br/>
